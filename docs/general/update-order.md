---
title: Update Order Information
---

Modul ini terdapat di ```/general/general-tools```. Jika user di SignalBit mengubah nama dari suatu Order atau nama color atau panel atau sebagainya. Maka perlu juga disinkronisasi ulang dengan data yang terlanjur terdaftar di NDS. Untuk itu digunakan Tool berikut : 

## Update General Order Info

User perlu **memilih suatu Order / WS** yang akan disinkronisasi. Lalu klik **UPDATE** untuk mengonfirmasi aksi sinkronisasi.

![Home](/assets/images/update-general-order.png) 

## Update General Order Color

User perlu **memilih suatu Order / WS**, lalu **Color Asal** dan **Color Tujuan** yang akan diubah. Lalu klik **UPDATE** untuk mengonfirmasi aksi sinkronisasi. 

![Home](/assets/images/update-general-order-color.png) 

:::info 

**Color Asal / From** hanya menunjukkan color yang sudah ada di NDS.

:::
