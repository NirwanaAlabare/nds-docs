# Permulaan

Project NDS adalah project aplikasi berbasis website yang dibuat untuk tracking produksi garment. NDS saat ini berfokus pada modul :

- Marker
- Cutting
- Stocker
- DC
- Laporan & Tools untuk Sewing
- Gudang
- Packing
- PPIC
- Mutasi Mesin
- Dasbor

### Stacks

Stacks yang sedang digunakan dalam project ini adalah sebagai berikut

- PHP 7.4 
- [Laravel 8](https://laravel.com/docs/8.x) 
- [Livewire v2](https://laravel-livewire.com/docs/2.x/quickstart) 
- MySQL

### Rekomendasi Software

Software yang sedang digunakan saat ini : 

- XAMPP Server v3.3
- Visual Studio Code 
- Firefox 
- Navicat
- Github Desktop
- PDI (Pentaho Data Integration)
